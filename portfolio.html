Portfolio

  • Arcade Game Project

    Implemented a version of the retro arcade game Joust in Java. I worked with a team to implement graphics features including movement, enemy tracking, and collision detection.

  • Editor Trees

    Built a height-balanced "AVL tree" (really a balanced-tree implementation of Lists) in Java with a partner. The tree can handle addition and deletion of nodes, and perform single and double rotations if necessary.

  • Guitar Hero

    Constructed a version of the video game Guitar Hero playable on an FPGA board using Quartus and Verilog. The game uses a 7-segment display to simulate the notes falling on the screen.

  • Linear Lights Out

    Used HTML, CSS, and JavaScript to implement a simple version of the Lights Out game. The goal of the game is to get all the lights to match by pressing on the lights, inverting it and its neighbors.
